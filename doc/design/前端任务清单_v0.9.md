# Agent Team Builder - 前端任务清单 v0.9

> 目标：按页面与组件落地前端实现（含模型库与主辅分配）

---

## Sprint 1：框架与基础骨架

1. 应用壳层
- 布局、导航、权限守卫。
- 路由新增 `/models` 系列页面。

2. 公共组件
- `AppTable/AppForm/AppModal/AppDrawer/AppConfirm`。

3. Store 初始化
- `useModelStore`。
- `useBootstrapStore` 增加角色 Agent 分配状态。

验收：
- 路由可访问，权限控制生效。

---

## Sprint 2：Model 库与 Agent 模型绑定

1. Model 列表页 `/models`
- 列表、筛选、状态标记、健康状态。

2. Model 详情页 `/models/:id`
- 配置展示、健康检查、发布按钮。

3. Agent 页面增强 `/agents`
- 增加默认模型选择器。
- 展示模型 tier 标签。

验收：
- 可以完成模型创建、发布、绑定 Agent。

---

## Sprint 3：项目启动向导主辅分配

1. 启动向导步骤3
- 新增 `RoleAgentAssignmentEditor`。
- 支持同角色多 Agent 列表。
- 支持设置 1 个 `PRIMARY` + 多个 `ASSISTANT`。

2. 失败反馈展示
- 组件 `BootstrapErrorList` 支持模型相关错误码。

3. 交互保护
- 未设置主负责人不可进入下一步。

验收：
- 1主2辅配置可以提交。
- 非法配置前端可拦截并提示。

---

## Sprint 4：看板与审计增强

1. 看板增强
- 角色下展示主负责人和助手。
- 展示当前模型与档位。

2. 审计页增强
- 支持 `MODEL` 与 `role-agents` 变更筛选。

3. 联调修复
- 错误码映射统一。
- 加载态、空态、异常态补齐。

验收：
- 主辅分配结果在看板可见。
- 审计可追踪模型与分配变更。
