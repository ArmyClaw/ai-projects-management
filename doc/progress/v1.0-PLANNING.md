## 开发规范

### 代码规范

```markdown
## 代码规范
- TypeScript strict模式
- ESLint + Prettier
- 组件命名 PascalCase
- 函数命名 camelCase
- 常量命名 UPPER_SNAKE_CASE
```

### Git提交规范

```markdown
## 提交规范
feat: 新功能
fix: Bug修复
docs: 文档更新
refactor: 重构
test: 测试
chore: 构建/工具
```
