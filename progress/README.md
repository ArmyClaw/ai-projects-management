## 技术决策记录

### 已确认决策

| 决策项 | 选项 | 决策 | 理由 |
|--------|------|------|------|
| 前端框架 | Vue 3 / React | Vue 3 | 模板语法AI友好 |
| UI组件库 | Naive UI / Element Plus | Naive UI | 主题配置灵活 |
| 后端框架 | NestJS / Fastify / Express | Fastify | 高性能、简单 |
| 数据库 | PostgreSQL / MySQL | PostgreSQL | JSON支持好 |
| Skill格式 | JSON Schema / YAML | JSON Schema | TypeScript原生支持 |
| 变量语法 | Handlebars / Mustache | Handlebars | 直观 |
| Git验证 | OAuth / Token / 不验证 | 不验证 | 最简方案 |
| 支付方式 | USDT / 支付宝 / 积分 | 纯积分 | 简化实现 |
| AIAgent | 系统账户 / 模拟用户 | 系统账户 | 便于管理 |

### 待讨论决策

无 - 所有核心决策已确认
