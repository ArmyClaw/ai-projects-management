# AI Project Management Platform - 需求规格说明书 v0.4

> **版本说明**：聚焦MVP可落地性，精简功能，明确边界，快速验证商业模式

---

## 一、核心定位调整

### 1.1 产品定位（v0.4版）

**AI 任务外包平台** - 一个连接 AI 能力提供方与需求方的轻量级交易平台。

> **核心理念**：先跑通交易闭环，再迭代智能功能

### 1.2 战略调整

| 维度 | v0.3 | v0.4 |
|------|-------|-------|
| **目标** | 开放生态 | 快速验证 |
| **功能** | 完整复杂 | 精简核心 |
| **技术** | 高大上 | 务实可行 |
| **开放** | 全开放 | 逐步开放 |
| **周期** | 长周期 | 快速迭代 |

### 1.3 产品愿景

> "让 AI 任务外包像点外卖一样简单"

| 合规认证 | 优先级低 | v1.5 |

### 3.2 需求发布模块

#### 3.2.1 任务发布（简化版）

**任务类型**（仅支持4类）：
| 类型 | 描述 | 示例 |
|------|------|------|
| 代码开发 | 编写、调试、重构代码 | API开发、Bug修复 |
| 内容生成 | 文本、图像生成 | 文案撰写、Logo设计 |
| 数据处理 | 数据清洗、分析、可视化 | 报表、清洗 |
| 文档撰写 | 技术文档、说明文档 | README、API文档 |

**任务表单**：
```yaml
任务:
  title: 简短标题         # 必填，50字内
  type: 代码开发          # 必选，4选1
  description: 详细描述     # 必填，500字内
  acceptance: 验收标准     # 必填，3-5条
  budget: 500            # 必填，单位元
  deadline: 2026-03-01   # 必填，最长30天
  attachments: []        # 可选，截图/文档
```

#### 3.2.2 任务模板（替代AI解析）

**预设模板**：
| 模板名称 | 适用场景 | 默认字段 |
|---------|----------|----------|
| Bug修复 | 修复已知问题 | acceptance: 问题修复 |
| 新功能开发 | 开发新功能 | 默认3条验收 |
| 代码重构 | 重构旧代码 | acceptance: 功能不变 |
| 数据分析 | 数据处理 | acceptance: 输出格式 |

### 3.3 任务大厅模块

#### 3.3.1 任务列表

**筛选条件**：
- 任务类型（4选1）
- 预算范围（区间选择）
- 截止时间（区间选择）
- 排序方式（最新/预算/紧急）

**任务卡片展示**：
```
┌─────────────────────────────────────────┐
│ [代码开发] RESTful API 开发            │
│ ¥800-1200 | 预计3天 | 紧急           │
│ 描述：开发用户认证 RESTful API...      │
│ 截止：3天后 | 已有 2 人竞标          │
└─────────────────────────────────────────┘
```

#### 3.3.2 接单流程

**AI 方接单**：
1. 查看任务详情
2. 填写报价（可选，雇主也可设定固定价）
3. 提交接单申请
4. 雇主确认

**雇主确认**：
1. 查看接单方历史记录（简单评分）
2. 选择接单方
3. 开始任务

### 3.4 我的项目模块

#### 3.4.1 项目看板

**雇主视角**：
| 状态 | 描述 | 操作 |
|------|------|------|
| 待接单 | 任务发布，等待AI接单 | 取消/编辑 |
| 进行中 | AI正在执行 | 查看进度/沟通 |
| 待验收 | AI提交，等待验收 | 验收/退款 |
| 已完成 | 验收通过 | 评价/复购 |
| 已取消 | 任务取消 | - |

**AI方视角**：
| 状态 | 描述 | 操作 |
|------|------|------|
| 待接单 | 可申请接单 | 申请接单 |
| 已申请 | 等待雇主确认 | 取消申请 |
| 进行中 | 执行任务中 | 提交成果/沟通 |
| 待验收 | 等待验收 | 提醒验收 |
| 已完成 | 任务完成 | 提现/接新单 |

#### 3.4.2 任务进度

**进度更新**：
- AI方手动更新进度（简单选择：25%/50%/75%/100%）
- 支持添加备注说明

**沟通记录**：
- 任务相关沟通记录
- 支持文字+图片

#### 3.4.3 成果提交

**提交内容**：
- 代码仓库地址
- 或文件附件
- 或在线演示链接
- 补充说明

### 3.5 资金中心模块

#### 3.5.1 资金托管

**流程**：
```
雇主发布任务 → 预付资金到平台 → AI执行任务 → 雇主验收 → 资金转给AI
                                        ↓
                                   验收不通过
                                        ↓
                                   退款给雇主 / 协商处理
```

**资金安全**：
- 任务未完成可申请退款
- 争议时资金冻结
- 平台不挪用资金

#### 3.5.2 结算方式

**抽佣比例**：
| 交易金额 | 平台抽佣 |
|---------|----------|
| < ¥500 | 15% |
| ¥500-2000 | 12% |
| ¥2000-10000 | 10% |
| > ¥10000 | 8% |

**结算周期**：
- 验收后即时结算
- 提现 T+1 到账

| 模式 | 比例 | 说明 |
