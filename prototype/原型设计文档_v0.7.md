# AI Project Management Platform - 原型设计文档 v0.7

> **版本说明**：基于v0.7需求规格设计的原型，涵盖MVP核心页面和交互流程

---

## 一、原型设计概述

### 1.1 设计目标

```
MVP原型设计：
├─ 覆盖核心用户流程
├─ 展示页面结构和交互
├─ 便于开发实现参考
└─ 可用于用户测试
```

### 1.2 角色与页面映射

| 角色 | 核心页面 | 说明 |
|------|----------|------|
| **发起人** | 项目列表 | 查看所有发起项目 |
| | 项目详情 | 查看项目进度、参与者 |
| | 发起项目 | 创建新项目 |
| | 验收管理 | 审核参与者提交 |
| **参与者** | 任务大厅 | 浏览可认领任务 |
| | 任务执行 | 使用Skill执行任务 |
| | 成本看板 | 查看API成本 |
| | 个人档案 | 查看评分、积分 |
| **通用** | 登录注册 | 用户认证 |
| | 首页 | 平台概览 |

### 1.3 原型页面清单

```
MVP核心页面（12个）：
├─ 1. 首页
├─ 2. 登录/注册
├─ 3. 发起人-项目列表
├─ 4. 发起人-发起项目
├─ 5. 发起人-项目详情
├─ 6. 发起人-验收管理
│
├─ 7. 参与者-任务大厅
├─ 8. 参与者-任务执行
├─ 9. 参与者-成本看板
├─ 10. 参与者-个人档案
│
└─ 11. 通用-Skill配置
└─ 12. 通用-结算记录
```
